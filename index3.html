Разделы

Интерактивный информационный справочник ИБ

Это упрощённый, но уже рабочий продукт. Система предназначена для локального использования, поддерживает разделение по категориям, поиск по записям и просмотр связанной информации. Структура сделана так, чтобы её можно было постепенно расширять в будущем.

Управление записями

Связанные записи: удерживайте Ctrl, чтобы выбрать несколько пунктов.
Новая запись будет добавлена в текущий раздел. Данные сохраняются локально в браузере.
Режим: добавление новой записи.

Новостная лента

Новости доступны для просмотра. Добавление новостей разрешено только администратору.

Авторизация

Для доступа к функциям управления записями и новостями выполните вход под учётной записью администратора.

Учебная учётная запись администратора используется для имитации системы авторизации в локальном прототипе.
Введите логин и пароль администратора.
